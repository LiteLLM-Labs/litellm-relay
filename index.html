LiteLLM Relay

Install a local proxy that shows intercepted AI traffic.

Relay runs on your machine, serves a PAC file, tunnels HTTPS traffic, and exposes a local shadcn-style dashboard at http://127.0.0.1:4142/.

Open local dashboard GitHub repo 
curl -fsSL https://raw.githubusercontent.com/BerriAI/litellm-relay/main/install.sh | bash

# Manual pilot: route Notion through Relay on Wi-Fi
curl -fsSL https://raw.githubusercontent.com/BerriAI/litellm-relay/main/install.sh \
  | bash -s -- --set-system-proxy "Wi-Fi"

What it does now

01

Runs a local CONNECT proxy on 127.0.0.1:4142.

02

Serves a PAC file that routes Notion domains through Relay.

03

Logs intercepted host, port, duration, byte counts, and shadow status.

04

Shows those events in a local dashboard with filters and raw JSON inspection.

curl -I -x http://127.0.0.1:4142 https://www.notion.so

Default mode does not decrypt TLS or read prompts. It captures proxy metadata and can send redacted shadow events through LiteLLM Gateway.
